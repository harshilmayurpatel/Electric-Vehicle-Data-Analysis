# Electric Vehicle Market Analysis Dashboard

## Project Overview
This project analyzes electric vehicle adoption trends using Tableau. 
The dashboard provides insights into EV growth, manufacturer dominance, 
and geographic distribution across the United States.

## Tools Used
Tableau
Excel / CSV Dataset
Data Visualization

## Key Insights
• EV adoption has grown rapidly after 2018
• Tesla dominates the EV market share
• Certain states lead EV adoption due to policy incentives
